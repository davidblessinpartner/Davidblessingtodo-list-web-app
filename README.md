# Responsive To-Do List Web App

A simple, responsive to-do list web application built using **HTML, CSS, and JavaScript**.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Tasks persist using browser localStorage
- Responsive design for desktop and mobile

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- localStorage API

## Purpose
This project demonstrates my ability to:
- Build functional web applications
- Manipulate the DOM using JavaScript
- Implement persistent client-side storage
- Write clean, readable front-end code
